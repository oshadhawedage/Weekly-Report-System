# Weekly Report Generator & Team Dashboard

A full-stack web application that allows team members to submit weekly reports and managers to analyze team progress through a dashboard.

## Tech Stack

- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: PostgreSQL (planned)
- Auth: JWT (planned)

## Project Structure

/frontend - React app
/backend - Express API

## Setup Instructions

### Frontend

```bash
cd frontend
npm install
npm run dev
```
